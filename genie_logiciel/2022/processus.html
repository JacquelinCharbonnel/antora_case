Exemple :

$ myeditor monprog.f90 (1)
$ gfortran -o monprog monprog.f90 (2)
$ ./monprog (3)
1 codage des instruction Fortran dans un fichier source monprog.f90
2 compilation du fichier source pour obtenir un binaire monprog
3 exécution du binaire

Dans l’exemple ci-dessus, gfortran est le nom du compilateur Fortran, monprog.f90 est le nom du programme source, et monprog est le nom souhaité pour le binaire.

Le binaire est propre à un architecture donnée (par exemple, un binaire généré pour Unix ne fonctionne pas sous Windows, et inversement). Un binaire généré sur une plateforme n’est pas forcément destiné à s’exécuter sur cette même plateforme. Par exemple, il est possible de générer un binaire pour Windows sous Unix : c’est de la cross compilation (réalisée par un cross compilateur).
