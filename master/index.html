L’objectif de ce site est d’expliquer comment le générateur de site Antora peut réaliser un support de cours de ce type.

Ce site est factice, mais il pourrait être le support de 4 cours ( Programmation C, Apache, Génie logiciel, Python Objet ) destinés aux étudiants de master Data Science. Chacun de ces 4 cours se décline en différentes versions : 2021, 2022, 2023, en français et en anglais, etc. Et sur le même principe, on imagine qu’il existe ailleurs un autre site de support de cours, destiné à une autre promotion (par exemple la licence informatique), avec des cours en commun entre les 2.

Comment utiliser Antora pour réaliser cela ? Pour le savoir plus, voir le making of.
