Ce site est factice. Il pourrait être le support de 4 cours ( Programmation C, Apache, Génie logiciel, Python Objet ) destiné aux étudiants de master Data Science. Chaque cours peut se décliner en différentes versions : 2021, 2022, 2023, en français et en anglais, etc. Et sur le même principe pourrait être réalisé un site de support de cours destiné à une autre promotion (par exemple la licence informatique), avec éventuellement des cours en commun entre les 2.

L’objectif de ce site est d’expliquer comment utiliser le générateur de site Antora pour réaliser cela. Pour en savoir plus, voir le making of.
