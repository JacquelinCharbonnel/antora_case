Presentation of the context

Website generator

A website generator is a program that transforms a tree of source files into a tree of HTML pages with one or more navigation systems to browse its pages. With such a generator, a webmaster creates a website without writing HTML pages, and without managing navigation links between pages. The generated site is static, therefore durable and robust over time, as it is insensitive to the security problems of motorized CMS.

Antora

Antora is a website builder which natively absorbs source pages in AsciiDoctor format to generate HTML pages. It is based on node.js.

AsciiDoctor

Asciidoctor is both a page description format and an engine for converting pages written in this format into HTML (and PDF) pages. The AsciiDoctor format is an extension of the AsciiDoc format, and the AsciiDoctor engine is a particular implementation of the AsciiDoc edit string. There are 2 equivalent implementations of the AsciiDoctoc engine, one in Ruby and the other in JavaScript. Antora uses Javascript implementation.

AsciiDoc

Asciidoc is a format for describing textual documents semantically rich, suitable for writing books and articles. Asciidoc is not interested in the rendering of documents, but in their composition. It is an evolution of the DocBook format, an XML dialect from the 1990s with an editing chain allowing HTML, PDF, postscript, text, epub, manpages, etc. versions to be generated from the same set of documents.

The couple (AsciiDoctor-Antora) therefore makes it possible to generate a website composed of potentially numerous and structurally complex pages.
