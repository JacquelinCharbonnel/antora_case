{
  int *c ;

  c = a ;
  a = b ;
  b = c ;
}

int x=1, y=2 ;
swap(&x,&y) ; // x vaut toujours 1 et y toujours 2

Cette fonction swap n’a toujours aucun effet

version 3 erronée
void swap(int *a, int *b)
{
  int *c ;

  *c = *a ; // néfaste, car c n'est pas initialisé
  *a = *b ;
  *b = *c ;
}

int x=1, y=2 ;
swap(&x,&y) ;

Cette fonction swap est néfaste, car risque de provoquer à un core dump. Pire, suivant la valeur indéterminée de c, on peut obtenir de temps en temps le bon résultat !!

version 4 correcte
void swap(int *a, int *b)
{
  int c ;

  c = *a ; // néfaste, car c n'est pas initialisé
  *a = *b ;
  *b = c ;
}

int x=1, y=2 ;
swap(&x,&y) ; // x vaut 2 et y 1
