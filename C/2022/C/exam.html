Examen

Exercice 1

Ecrire un programme qui donne la liste des utilisateurs définis sur le système, et pour chacun d’eux le nom de leur groupe principal

fonctions utiles : getpwent, getgrgid
Exercice 2

Ecrire un programme C qui récupère 2 chaines de caractères en argument sur la ligne de commande, supposées être un nom de fichier et un nom d’utilisateur, et qui renvoie un code égal à :

  • 4 si le nombre d’arguments fournis n’est pas 2, sinon

  • 3 si le fichier n’existe pas, sinon

  • 2 si l’utilisateur n’existe pas, sinon

  • 1 si le fichier n’appartient pas à l’utilisateur, sinon

  • 0

  • fonctions utiles : getpwnam, lstat

  • pour afficher, depuis un terminal, le code de retour du dernier process lancé :

    $ echo $?
Exercice 3

Ecrire un programme qui donne la liste des groupes d’utilisateurs définis sur le système, avec pour chacun d’eux la liste de leurs membres

fonction utile : getgrent
