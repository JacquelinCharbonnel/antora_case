Exercice {exo_prefix}1

Soit la classe ci-dessous :

class Multiples():
    def __init__(self,k):
        self._k = k
        self._i = 0

    def courant(self):
        self._i += 1
        return self._k*self._i

Elle permet d’obtenir une liste de multiple. Par exemple pour obtenir les 10 premiers multiples de 7 :

m = Multiples(7)
for i in range(10):
    print(m.courant(), end=" ") # 7 14 21 28 35 42 49 56 63 70
Question {exo_prefix}1.1

Sachant que préfixer un attribut par un _ signifie par convention que l’attribut est privé, comment utiliser la classe Multiple pour sortir tous les couples formés des 10 premiers multiples de n. Par exemple pour n=7 :

(7,7) (7,14) (7,21) ... (14,7) (14,14) (14,21)...
Question {exo_prefix}1.2

Pour pouvoir effectuer simultanément plusieurs itérations sur cette classe, on va sortir la gestion de l’itération de la classe Multiples.

Créer un début d’itérateur permettant d’écrire :

m = Multiples(7)
it1 = It(m)  # 1 premier itérateur pour la boucle externe
m1 = next(it1)
for i in range(10):
  it2 = It(m)  # 1 second itérateur pour la boucle interne
  for j in range(10):
    print(f"({m1},{next(it2)})", end=" ")
  m1 = next(it1)

pour afficher les couples de multiples.

m = Multiples(7)

Question {exo_prefix}1.5

Pour éviter de polluer l’espace de nommage global, il est d’usage que l’itérateur (qui est finalement une classe invisible aux utilisateurs) soit une classe interne à la classe itérée. Modifier le code en conséquence.
