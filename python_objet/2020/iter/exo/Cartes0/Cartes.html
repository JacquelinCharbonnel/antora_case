Exercice 1

Soit la classe Carte modélisant une carte d’un jeu de 32 :

class Carte():
  def __init__(self,v):
    self.valeur = v

  def figure(self):
    return self.valeur not in list(range(7,11)) + ['As']

  def __str__(self):
    return f"{self.valeur}"

  def __repr__(self):
    return f"{self.valeur}"

Créer un type indiçable Jeu s’utilisant comme :

  mon_jeu = Jeu()
  for carte in mon_jeu:
    print(f"{carte} {carte.figure()}")

pour donner :

7 False
8 False
9 False
10 False
Valet True
Dame True
Roi True
As False

Vérifier que l’on peut bien indicer un objet de type Cartes. Peut-on prendre une tranche de Cartes ?

Ecrire un programme qui sort tous les couples de cartes possibles : (Valet,Valet) (Valet,Dame), (Valet,Roi), etc…​
