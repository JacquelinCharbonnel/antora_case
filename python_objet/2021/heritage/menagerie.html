Ménagerie

le dauphin siffle
les mammifères marchent et s'expriment de façon très diverses
l'épervier piaule
poules et pintades sont des gallinacées
l'autruche claquette et ne vole pas
le rossignol est un oiseau
la chauve-souris grince
le saumon est un poisson
les poissons généralement nagent et n’émettent pas de cri
le chat miaule
l'exocoetidae vole
les gallinacées gloussent
le poisson-crapaud grogne
les oiseaux en général volent et sifflent

Question 1

Accepter, les yeux fermés, les affirmations ci-dessus. Les modéliser sous forme d’une hiérarchie de classes. Doter ces classes de méthodes définissant leur comportement (leur déplacement et leur expression).

Modéliser le comportement par des méthodes (par exemple se_deplace() et s_exprime()) renvoyant une simple chaine de caractères (par exemple "je galope", "je rugis").

Question 2

Ajouter les digitigrades dans la hiérarchie précédente.

Question 3

On souhaite pouvoir attribuer à chaque instance un nom (Polux, Zouzou, Cheetah…​). Une instance doit aussi être capable de se présenter (sur un print()), c’est-à-dire donner son nom, le nom de sa classe, et le nom de sa (ou ses) super-classe directe.

Modifier les classes en conséquence.

Question 4

Enfin, pour terminer la conception des classes, définir, pour chaque animal, un nouveau comportement : la fuite, qui consiste à se déplacer en s’exprimant.

Question 5

La conception des classes étant terminée, il s’agit maintenant de les utiliser. Pour cela, des instances vont être créés dans un programme principal.

L’idéal est de mettre le programme principal dans fichier source .py séparé, et d’y importer le fichier de définition des classes.

A défaut, le programme principal peut être placé après les définitions de classes, dans le même fichier, précédé d’un test if __name__=="__main__":.

Dans ce programme, créer une ménagerie (une liste d’instances), puis présenter chaque animal, et enfin effrayer la ménagerie.
