Exercice xx2

Soit la classe Carte modélisant une carte d’un jeu de 32 :

class Carte():
  figures = {'Valet','Dame','Roi'}
  honneurs = figures.union({'As'})
  valeurs = set(range(7,11)).union(honneurs)
  couleurs = {'Coeur','Carreau','Pique','Trèfle'}

  def __init__(self,v,c):
    if not v in Carte.valeurs: raise ValueError("valeur incorrecte")
    if not c in Carte.couleurs: raise ValueError("couleur incorrecte")
    self.couleur = c
    self.valeur = v

  def figure(self):
    return self.valeur in Carte.figures

  def __str__(self):
    return f"{self.valeur} de {self.couleur}"

  def __repr__(self):
    return f"{self.valeur} de {self.couleur}"

Créer un type indiçable Coeurs s’utilisant comme :

les_coeurs = Coeurs()
for carte in les_coeurs:
  print(f"{carte} -> {carte.figure()}")

pour donner :

7 de Coeur -> False
8 de Coeur -> False
9 de Coeur -> False
10 de Coeur -> False
Valet de Coeur -> True
Dame de Coeur -> True
Roi de Coeur -> True
As de Coeur -> False

Vérifier que l’on peut indicer un objet de type Cartes. Peut-on prendre une tranche de Cartes ?

Ecrire un programme qui sort tous les couples de cartes possibles : (Valet,Valet) (Valet,Dame), (Valet,Roi), etc…​
